# NACRE//ORCHID Agent Card

## Mission

Transform NACRE//ORCHID from visual-system intelligence into a small, coherent, production-shaped Next.js vertical slice.

## Governing law

**Make the site feel like opal, but architect it like a newsroom.**

## Non-negotiables

- Meaning before surface.
- Tokens before components.
- Components before screens.
- CSS/SVG before WebGL.
- Accessibility before spectacle.
- QA before expansion.
- Every visual effect needs a semantic job.
- No ornament without hierarchy.
- No hover-only UX on mobile.
- No motion without reduced-motion fallback.

## Build only

- Home page
- Gallery page
- Artifact detail modal
- Studio preview section
- Shared navigation/layout
- Design-token CSS system
- Seed artifact data
- Responsive layout
- Reduced-motion and reduced-transparency fallbacks

## Defer

- Auth
- Database
- Marketplace
- Full community
- Video system
- Audio system
- Heavy WebGL
- AETHER dependency

## Core semantic mapping

- Pearl = saved, memory, completion, focus.
- Orchid = creation, bloom, transformation.
- Lace = veil, preview, modal, hidden layer.
- Gold = syntax, boundary, premium state, selection.
- Abalone = artifact depth, gallery tile, active surface.
- Opal = responsive intelligence, hover, listening, success.
- Labradorite = dark mineral field, background depth.
- Black lacquer = stage, contrast, atmosphere.

## Done means

The app runs, TypeScript passes, mobile works, keyboard focus is visible, gallery filters work, modal opens/closes accessibly, reduced-motion fallbacks exist, and the visual identity reads as NACRE//ORCHID within three seconds.
