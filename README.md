# NACRE//ORCHID

A luxury generative-art and design-system platform where pearl, orchid, lace, abalone, opal, labradorite, and gold cloisonné become product architecture, motion grammar, symbolic UX, and responsive storytelling.

## Governing law

**Make the site feel like opal, but architect it like a newsroom.**

## First production target

The first build is a focused vertical slice:

1. Home page
2. Gallery page
3. Artifact detail modal
4. Studio preview section
5. Shared navigation/layout
6. Design-token CSS system
7. Seed artifact data
8. Responsive layouts
9. Reduced-motion and reduced-transparency fallbacks

## Non-goals for v1

Do not build auth, database, marketplace, full community, audio engine, video engine, or WebGL-heavy scenes yet.

## Core stack

- Next.js
- TypeScript
- CSS variables / token classes
- Framer Motion for component motion
- Radix UI for modal/dialog behavior where useful
- Local seed artifact data
- SVG filters and CSS shader surfaces first

## Core components

- ShellRoot
- CursorLight
- SiteNav
- PearlButton
- TokenChip
- SemanticMeter
- NacreCard
- AbaloneTile
- LaceModal
- ReliquaryHero
- ArtifactGrid
- StudioPreview

## Quality bar

The build should be accessible, responsive, visually distinctive, tokenized, production-shaped, and small enough to extend without becoming fragile.

## Key rule

No ornament without hierarchy. No iridescence without material logic. No motion without semantic purpose.
